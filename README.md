# STM32 Ultrasonic Distance Alert System

An embedded firmware project implementing a real-time ultrasonic distance alert system using the STM32F103 microcontroller.

## Features
- Ultrasonic distance measurement using timer input capture
- Interrupt-driven, non-blocking firmware design
- OLED display over I2C for real-time distance visualization
- PWM-controlled buzzer for proximity alerts
- Button-based user interaction (mute / mode control)

## Hardware Used
- STM32F103 (Blue Pill)
- Ultrasonic Sensor (HC-SR04 or equivalent)
- OLED Display (I2C)
- Buzzer
- Push Buttons

## Development Environment
- STM32CubeIDE
- STM32 HAL Drivers
- ARM Cortex-M3
